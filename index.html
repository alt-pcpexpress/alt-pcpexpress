Introduction

Welcome to New MyPCP Express Application! This application is designed to streamline the process of managing courier packages, from creation to delivery. It offers a user-friendly interface and robust backend to ensure efficient handling of package logistics.

Features

  • Package Management: Create, update, and track packages.
  • User Authentication: Secure login for couriers.
  • Real-time Tracking: Track packages in real-time.
  • Outlet Location: See the addresses of PCP Express outlets and office agents.
  • Check Postage: allows users to check postage in the application based on coverage area

Installation

Prerequisites

  • Flutter (v3.x or later)
  • ASP.NET Core (v5.0 or later)
  • SQL Server

Steps

  1. Clone the repository

    git clone https://github.com/latiefdole/courier-expedition-package.git
cd myPCP Express

  2. Frontend Setup

    • Navigate to the myPCP Express directory 
      cd myPCP Express
    • Install dependencies 
      flutter pub get
    • Run the Flutter application 
      flutter run

Technologies Used

  • Frontend: Flutter
  • Backend: ASP.NET Core
  • Database: SQL Server
